# Docs Reorg Notes

This zip reorganizes the docs for agent/Codex usage.

## Main changes

- Moved `AGENTS.md` to repo root.
- Moved agent-specific docs into `docs/agent/`.
- Moved proposal material into `docs/proposals/`.
- Moved older/status/checklist/history-heavy docs into `docs/archive/`.
- Split `docs/WORK_LOG.md` into `docs/WORK_LOG_RECENT.md` and `docs/archive/WORK_LOG_FULL.md`.
- Rewrote `docs/DOCS_INDEX.md` so the active read path is smaller and clearer.

## Intent

The goal is not to delete knowledge. The goal is to separate active instructions/contracts from historical or proposal context so Codex has a cleaner default reading path.
